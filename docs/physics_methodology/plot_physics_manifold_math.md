# Phase Manifold Visualizations

The visualization scripts plot the integrated EoS curves to visually verify the physics.

The generated charts usually project the data into two main topologies:
- The Mass-Radius plane, which shows the physical size of the stars and the obvious kink characteristics of different phase transitions.
- The Mass-Deformability plane, which plots $\Lambda$ against $M$ to show how tidal response decays as the star gets heavier and more compact.

These plots are used mostly as a sanity check to confirm the generators are producing valid, separated populations before passing the data into the machine learning pipelines.
